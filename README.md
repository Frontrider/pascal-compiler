# pascal-compiler

This is a **Pascal** compiler written in Java.

## How do I compile?
1. Compile emulator: `javac Emulator.java`
2. Give the compiler a Pascal file as input: `java Emulator name_of_file.pas`

For example: 
`java Emulator ../examples/array.pas`
